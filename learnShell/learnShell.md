# 常用shell命令

## 1.常见shell

```shell
ls - 列出目录内容
cd - 切换目录
pwd - 显示当前工作目录
mkdir - 创建新目录
cp - 复制文件或目录
mv - 移动或重命名文件或目录
rm - 删除文件或目录
touch - 创建空文件
cat - 显示文件内容
head - 显示文件开头部分
tail - 显示文件末尾部分
echo - 输出文本
which - 显示命令所在路径
find - 查找文件或目录
grep - 搜索文本
chmod - 更改文件权限
ln - 创建链接
df - 显示文件系统磁盘空间使用情况
du - 显示目录或文件磁盘使用情况
mount - 挂载文件系统
umount - 卸载文件系统
ps - 显示进程状态
top - 实时显示系统运行状况
kill - 终止进程
jobs - 显示当前作业
nohup - 运行不受挂断信号影响的命令
at - 定时执行命令
who - 显示当前登录用户
su - 切换用户身份
sudo - 以超级用户权限运行命令
tar - 文件打包与解包
zip - 创建压缩文件
unzip - 解压缩文件
scp - 通过SSH复制文件
ssh - 远程登录
sftp - 安全文件传输
ftp - 文件传输协议
wget - 下载文件
curl - 数据传输工具
ping - 网络连接测试
traceroute - 路由跟踪
netstat - 网络状态
ifconfig - 配置网络接口
ip - 显示和操作网络接口
route - 显示和操作路由表
```
