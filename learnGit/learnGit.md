# 把这些命令扔进gpt让它举例子给你

## git

```shell
git config --list # 显示当前配置

git config --global user.name "Your Name" # 设置全局用户名

git config --global user.email "your.email@example.com" # 设置全局用户邮箱

git init # 初始化一个新的Git仓库

git clone https://github.com/user/repository.git # 克隆一个现有的Git仓库

git status # 检查当前仓库的状态(查看哪些修改没提交)

git add filename # 添加文件到暂存区

git commit -m "提交信息" # 提交暂存区的文件到仓库
git commit --amend # 想要修改一下

git push origin branch-name # 将本地分支推送到远程仓库

git rebase
git merge
git pull --rebase
git restore
```
